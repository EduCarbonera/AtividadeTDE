Olá professor! para utilizar meu programa, siga esse processo simples;
Após utilizar o RUN da sua IDE de preferência, o programa irá requisitar o input, de qual txt o senhor
deseja abrir. Para testagem, já adiantei os arquivos de: "teste1.txt", "teste2.txt" e "teste3.txt"

O programa suporta ler qualquer nome de arquivo, desde que consiga localizar seu caminho, e que o arquivo 
esteja formatado corretamente, no padrão estipulado pela rubrica e em formato de texto(.txt)

Para a lógica utilizada no código, ele funciona pelo seguinte meio:
	após a leitura e processamento dos dados compostos no arquivo TXT, ele realizará um loop
	simples, identificando: quantas operações, o operador e os conjuntos.

	resultando assim, em cada valor único registrado pelo terminal.
	feito isso, ele dá um pulo no contador, para que não repita as operações passadas.
